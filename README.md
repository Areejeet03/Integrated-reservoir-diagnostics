# Integrated-reservoir-diagnostics
This project is an integrated, Python-based reservoir diagnostics and production optimization workflow that mathematically unifies dynamic fluid PVT behavior, multi-variable combination-drive material balance, nodal lifting performance, and empirical decline curve analysis (DCA) into a single, cohesive system. It operates by reading both reservoir constants and historical time-series data directly from a structured Excel workbook (reservoir_data.xlsx) . The Python pipeline sequentially calculates pressure-dependent fluid properties using Standing and Beggs-Brill correlations, executes a least-squares linear regression (Havlena-Odeh method) to history-match original oil in place (OOIP) and aquifer water influx, simulates bottomhole well flowing pressures and operating rates across multiple depletion stages using Vogel's IPR and the Darcy-Weisbach equation, and ultimately fits an Arps hyperbolic decline model to forecast 10-year reserves—automatically auditing and cross-validating the empirical decline forecast against the physical thermodynamic limits of the active drive mechanisms to guarantee a physically consistent result.
